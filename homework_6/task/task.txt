Домашнее задание 6

Task "Модульный калькулятор"

Создайте пакет calculation который содержит модули multiply,add, divide, subtraction 
Каждый модуль должен содержать, по функции названия которой совпадает с названием модулей.
Данные функции будут отражать математические операции такие как добавлние (add), отнимание (subtraction), умножение (multiply) и деление (divide)

далее реализуйте отдельно файл main.py который будет являтся запускаемым скриптом, для этого используйте условие if name == main...

В данном скрипте пользователь будет вводить 3 раза: 1 раз - 1ый операнд только число, 2ой раз это символ операции "+" "-" "/" "*"

3ий раз - 2ой операнд, также только число. реализуйте ввод ДО ТЕХ ПОР, пока пользователь не введет в 1ом и 3ьем вводе число и во 2ом случае ТОЛЬКО символ операции.

После ввода результат должен быть в виде: "<1ый операнд > <знак операции> <2ой операнд> равно <результат>".
Также пользователь должен не прекращать ввод и выход из программы пока он не напишет: exit.



Пример ввода и вывода:

введите 1 операнд: abc

abc

введите 1 операнд: 2

2

введите действие: 0

0

введите действие: abc

abc

введите действие: +

введите 2 операнд: lalala

lalala

введите 2 операнд: 3

2 + 3 равно 5

введите 1 операнд: exit

и ТОЛЬКО после этого ваше приложение закончит работу



Для выполнения вам понадобятся: циклы while, функции, условия, модули, ввод вывод
